## 프로젝트 설명
- lifesemantics사에서 만든 에필트래커 밴드를 착용하고 동국대학교 캠퍼스를 걸어다니면서 건물들 간의 거리 및 소모열량 데이터 수집 
- 수집한 데이터를 이용해서 학생들의 시간표에 따른 그날의 예상소모열량 예측
- 이동경로, 예상도착시간, 예상이동거리, 예상소모열량 제공
